Basic Alert

ax5dialog creates a dialog by extending the argument to be passed to the alert method to the attributes defined in setConfig, then delivered to the user.
how to call the dialog is, alert, confirm and prompt, we will introduce the basic use of alert in it. 

<script type="text/javascript">
    $(document.body).ready(function () {
        var dialog = new ax5.ui.dialog();
        dialog.setConfig({
            title: "AX5 Dialog"
        });

        $('#alert-open').click(function () {
            dialog.alert('Alert message');
        });
        $('#alert-close').click(function () {
            dialog.close();
        });
    });
</script>

In the example, we introduce the basic use to output a alert dialog using dialog.alert.
However, you can not completely stopping the operation of the application as window.alert in dialog.alert.
If after the alert, user clicks the [OK] button, you need to pass an anonymous function as the second argument of the alert if there is a need of any such actions.

Alert callBack

In the following example, after the alert, when the user clicks the [OK] button, run an anonymous function that is defined as the second argument, To "this" anonymous function, you pass multiple arguments required developers. 

<script type="text/javascript">
    $(document.body).ready(function () {
        var dialog = new ax5.ui.dialog();
        dialog.setConfig({
            title: "AX5 Dialog"
        });

        $('#alert-open-callback').click(function () {
            dialog.alert('Alert message', function () {
                console.log(this);
            });
        });
    });
</script>

To "this" anonymous function, has been included, such as the following objects. You can check directly open a console window. 

{key: "ok", value: Object, dialogId: "ax5-dialog-2", btnTarget: button.btn.btn-default}
