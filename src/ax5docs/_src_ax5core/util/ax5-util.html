2 ax5.util util

ax5.util.get_type

Return argument object type

ax5.util.get_type(1); // "number" ax5.util.get_type("1"); // "string" ax5.util.get_type([0,1,2]); // "array" ax5.util.get_type({a:1}); // "object" ax5.util.get_type(function(){}); // "function" ax5.util.get_type(document.querySelectorAll("div")); // "nodelist" ax5.util.get_type(document.createDocumentFragment()); // "fragment"

Javascript object type name is not clear. so util.get_type method very useful.

ax5.util.is_[type]

Return argument object type is [type] result.

// return is window. ax5.util.is_window( window ); // return is element. ax5.util.is_element( document.getElementById("#ax5-util-is-type") ); // return is Object. ax5.util.is_object(); // return is Array. ax5.util.is_array(); // return is Functon. ax5.util.is_function(); // return is String. ax5.util.is_string(); // return is Number. ax5.util.is_number(); // return is nodeList. ax5.util.is_nodelist( document.querySelectorAll(".content") ); // return is undefined. ax5.util.is_undefined(); // return is undefined|''|null. ax5.util.is_nothing();
