ax5.info 오브젝트에 대해 기술합니다

ax5.info.version

ax5의 버전변수 입니다.

ax5.info.event_keys

event_keys 오브젝트의 JSON 구조는 다음과 같습니다.

// ax5.info.event_keys { "BACKSPACE": 8, "TAB": 9, "RETURN": 13, "ESC": 27, "LEFT": 37, "UP": 38, "RIGHT": 39, "DOWN": 40, "DELETE": 46, "HOME": 36, "END": 35, "PAGEUP": 33, "PAGEDOWN": 34, "INSERT": 45, "SPACE": 32 }

ax5.info.browser

browser 오브젝트의 JSON 구조는 다음과 같습니다.

// ax5.info.browser {"name": "chrome", "version": "46.0.2490.86", "mobile": }

ax5 코드가 초기화 할 때마다 오브젝트가 생성됩니다.
